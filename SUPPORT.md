# Support

Thanks for using Settler OSS. The best way to get help is through community channels so answers are discoverable.

## Quick Help Checklist

Before opening a new thread, please:

1. Read the docs: [docs/INDEX.md](./docs/INDEX.md)
2. Search existing issues: https://github.com/shardie-github/settler-oss/issues
3. Search discussions: https://github.com/shardie-github/settler-oss/discussions
4. Review the FAQ: [FAQ.md](./FAQ.md)

## Community Support

- **Q&A and usage help:** https://github.com/shardie-github/settler-oss/discussions/new?category=q-a
- **Ideas and proposals:** https://github.com/shardie-github/settler-oss/discussions/new?category=ideas
- **General discussion:** https://github.com/shardie-github/settler-oss/discussions/new?category=general

## Bug Reports & Feature Requests

- **Report a bug:** https://github.com/shardie-github/settler-oss/issues/new?template=bug_report.yml
- **Request a feature:** https://github.com/shardie-github/settler-oss/issues/new?template=feature_request.yml

## Security Reports

Please report vulnerabilities privately using the process in [SECURITY.md](./SECURITY.md).
